About Me

As an experienced and versatile IT Specialist for the last 6 years working at Darose Control Systems Ltd, I’ve acquired a wealth of technical skills within the IT sector, with extensive technical support, network administration, software development and database management. During this time I have worked with several multi-million dollar corporate clients around the world, including those in the Dairy, Seafood, and lumber industry.

In my spare time, when I'm not enjoying the outdoors, you can usually find me doing many of my favorite hobbies including playing music, tinkering with electronics, and developing websites. In the past year I've really taken a liking to Node.js and the possibilities is has to offer in web development. On the front side of things, my weapons of choice are Photoshop/Inkscape for drawing layouts and Sublime Text to produce clean semantic code. Generally I prefer the development side of things, however I do like to experiment with new and exciting web layouts. In the past I’ve also used Flash for animations and entire web sites.

Currently, my CMS of choice is WordPress which is coded in php and although before I learned about Node.js (and Express.js), I did enjoy creating websites using the Django Framework (Python). This really helped me get a grasp on the Model-View-Controller software architectural pattern that's similarly used in many different frameworks like Express.js.

This website was created with most of my interests in mind. For starters, the site has a clean layout and will adapt to different screen sizes whether it be a phone, tablet or desktop. Next is the web server that I have written in Node.js. It is realtime meaning that if, for example, you click the color picker below, and choose a new color, everyone else who is currently viewing the page (across the world) will see the color immediately change as well. The reason why I did this was I wanted it to be the interface (controller) for an Arduino microcontroller that I have set up at my home in Halifax, Nova Scotia. An arduino (pictured below) is an electronic development board that you can plug sensors and motors into and control them through code.

Node.js

Node.js is at the heart of this website. It is an open source, cross-platform server environment for networked applications. In other words, it is a number of tools that allowg you to build applications in JavaScript. Within node.js, I've used a framework called Express that allows you to build applications specifically for the web. Another major JavaScript framework I've used within this project is Johnny-Five. It made it easy to interface with the Arduino. Lastly, another framework, called Socket.io was used to create real-time event-based communication between the website, to the Arduino.

Amazon AWS

Amazon AWS is a collection of remote computing services, also called web services, that make up a cloud computing platform offered by Amazon.com. Because of this, Amazon is Awesome! They allow you to run a virtual environment (windows or linux) in the cloud that you can use to do many different things with. A general example is using EC2 to store your website, VPC to organize your Virtual Private Networking Inferstructure, and S3 to store all of your static files (photos). They even have a domain system that you can now buy your domains from (ex bradlannon.ca). This is just the tip of the iceberg. Because of these new advances in Cloud computing, I no longer think 'bookstore' when I think of Amazon.
